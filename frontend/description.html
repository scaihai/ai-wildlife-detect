Description

WildlifeDetect is a production-grade object detection system for wildlife monitoring. It combines a custom fine-tuned Faster R-CNN model with production MLOps infrastructure to identify and localize Lion, Ostrich, and Oryx in images with high accuracy. The model was fine-tuned via transfer learning to handle the unique challenges of wildlife detection - class imbalance, occlusion, and pose variation - then deployed as a versioned, containerized service.

The system showcases end-to-end ML engineering: TensorFlow Serving handles inference at scale via REST and gRPC endpoints, while Prometheus and Grafana provide real-time observability into model latency, throughput, and system health. Docker Compose orchestrates the full stack with versioned models in a centralized registry, supporting production practices like canary deployment and rollback. This architecture demonstrates professional MLOps standards - reproducible, scalable, and cloud-ready.

Technologies Used

  • Model & Inference: Faster R-CNN fine-tuned on custom wildlife datasets, served via TensorFlow Serving.
  • Observability: Prometheus metrics + Grafana dashboards for real-time system monitoring.
  • Infrastructure: Docker & Docker Compose with centralized model registry for reproducible, cloud-ready deployment.
  • Frontend: Flask and JavaScript for direct REST API integration.
